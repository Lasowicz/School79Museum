# Школьный музей МАОУ «СОШ № 79» г. Уфа

Этот сайт - авторский проект. Первоначально задумывался, как простой лендинг, визитка школьного музея. 
Далее он перерос в небольшой портал, посвященный истории Великой Отечественной войны. Вероятно, далее он перерастет в полноценный музейно-образовательный проект.
Данный сайт решает основную проблему подобных музеев - отсутствие вовлеченности в информационное поле.

## Технологии

- **Верстка:** HTML5, CSS3 (Flexbox, Grid)
- **Интерактивность:** JavaScript (ES6+)
- **Адаптивность:** медиа-запросы для корректного отображения на мобильных устройствах и десктопах
- **Хостинг:** GitHub Pages

## Особенности

Сайт музея включает следующие разделы:

- **Главная** — приветственное слово, краткая информация о музее
- **История школы** — хронология развития учебного заведения, ключевые даты и события
- **Экспозиции** — виртуальные выставки, описание экспонатов
- **Наши достижения** — награды, дипломы, участие в конкурсах
- **Мероприятия** — календарь событий, отчёты о прошедших встречах и экскурсиях
- **Контакты** — информация для связи с кураторами музея

Дополнительно реализованы:
- адаптивное меню навигации;
- галерея изображений с возможностью просмотра в полном размере;
- плавная прокрутка к якорям;
- интерактивные элементы (подсветка активных пунктов меню, анимации при наведении).
